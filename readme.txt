== Description ==
This widget provides a form to lookup amateur radio call signs from the QRZ.RU site. QRZ.RU provides information about Russian amateur radio operators (also includes information from the QRZ.COM and some others call signs books). This widget was originally made by Tom Epperly for searching call signs on the QRZ.COM.

== Installation ==
1. Upload `qrzrusearch.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget in the sidebar using the editor for your theme.
